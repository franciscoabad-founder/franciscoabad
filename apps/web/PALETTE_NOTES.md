# Paleta: Indigo · Periwinkle

Base navy casi negra con acento periwinkle medio saturado. Evita el "startup tech" manteniendo el azul muy oscuro y el periwinkle como acento puntual, nunca fill de fondo.

## Tokens principales

## Cómo volver al Ember actual

git checkout feat/astro-migration

## Cómo cambiar a otra paleta

git checkout exp/palette-{nombre}
Disponibles: cobalt, iron, conifer, petrol, console, indigo
