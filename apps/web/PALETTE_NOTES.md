# Paleta: Console · Volt

Monocromático puro hasta el Volt. Un único punto de color, quirúrgico. Para alguien que sabe lo que hace y no lo decora.

## Tokens principales

| Rol | Token | HEX |
|-----|-------|-----|
| Background base | Pitch | #0A0A0A |
| Cards | Console | #0F0F11 |
| Bordes | Slate | #1A1A1F |
| Reglas secundarias | Wire | #3A3A42 |
| Captions | Ash | #6E6E78 |
| Texto secundario | Chalk | #D4D4D0 |
| Texto principal | Off-white | #F4F4F0 |
| Acento único | Volt | #C8E664 |

## Glitches conocidos

Por ser monocromático con acento neón, algunos hover/focus que
asumen un acento cálido pueden leer raros. Documentar al revisar
visualmente, no arreglar (es prueba A/B).

## Cómo volver al Ember actual

git checkout feat/astro-migration

## Cómo cambiar a otra paleta

git checkout exp/palette-{nombre}
Disponibles: cobalt, iron, conifer, petrol, console, indigo
