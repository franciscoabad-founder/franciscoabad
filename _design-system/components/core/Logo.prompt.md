The Francisco Abad wordmark, set in live Gotham type so it stays crisp and recolorable. FRANCISCO is the ExtraLight whisper, ABAD the Black protagonist.

```jsx
<Logo layout="horizontal" size={32} tone="dark" />
<Logo layout="stacked" size={56} tone="light" />
```

`tone="dark"` colors ABAD Ultramarine; `tone="light"` colors it Royal. For monochrome use over photos pass `abadColor="var(--white)"`.
