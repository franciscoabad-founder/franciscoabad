# Paleta: Petrol · Cool whites

Petróleo profundo con blancos fríos. Técnico sin ser frío, considerado sin ser lento. El espacio que pocas marcas ocupan con intención.

| Rol | Token | HEX |

| Texto secundario | Steel | #7A8FA0 |
| Acento principal | Petrol | #2E7080 |
| Acento darker | Petrol Deep | #1E5060 |
