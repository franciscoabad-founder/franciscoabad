# Pancho OS Desktop

App local de Windows para mantener `https://www.franciscoabad.com/os` abierta como ventana propia.

## Uso

```bash
npm install
npm start
```

## Empaquetar

```bash
npm run dist
```

El instalador queda en `release/`. Al instalarlo puedes fijarlo al taskbar. La app tambien queda en el tray y puede iniciar con Windows desde el menu del icono.
