Branded action button. Use `primary` (filled Ultramarine) for the main CTA on any view, `secondary` (outline) for the alternate, `ghost` for low-emphasis text actions.

```jsx
<Button variant="primary" size="lg" onClick={go}>Trabajemos juntos</Button>
<Button variant="secondary">Explora recursos</Button>
```

Variants: `primary` · `secondary` · `ghost` · `metric` (champagne fill, reserved · almost never a general CTA). Sizes: `sm` `md` `lg`. Radius is locked to 6px per brand spec. Pass `iconLeft` / `iconRight` for icons.
